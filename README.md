# Proyecto-7-database-library-project
El proyecto incluye el uso de tablas, relaciones y claves primarias y foráneas, así como un diagrama ERD que representa visualmente la estructura de la base de datos.

La base de datos permite gestionar información sobre: Lectores: datos personales y fecha de registro.
Libros: información de cada libro disponible en la biblioteca.
Préstamos: registro de los libros prestados, incluyendo fechas de préstamo y devolución esperada.

El proyecto demuestra mis habilidades en:

Modelado de datos relacional.
Creación de tablas en SQL.
Definición de claves primarias y foráneas.
Representación gráfica de la estructura de la base de datos mediante un ERD.
